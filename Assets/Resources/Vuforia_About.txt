The User Defined Targets sample shows how to capture and create an image target at runtime from user-selected camera video frames.

Key Functionality
- Create and manage user defined image target
- Activate Extended Tracking
- Manage camera functions: flash and continuous autofocus

Instructions
- Hold device parallel to feature rich target and tap camera button
- Single tap to focus
- Double tap to access options menu
